# Image Forgery Detection Using ELA & Multi-Scale CNN

A deep learning project for detecting forged images using Error Level Analysis (ELA) and a custom Multi-Scale CNN architecture.

## Dataset

CASIA 2.0 Dataset

## Results

- Accuracy: 92.08%
- Precision: 93.7%
- Recall: 93.7%
- F1 Score: 92.1%

## Features

- Error Level Analysis (ELA)
- Multi-Scale CNN
- Squeeze-and-Excitation Attention
- Inception-Style Convolution Blocks
- AdamW Optimizer
- Cosine Learning Rate Scheduling

## Technologies

- Python
- TensorFlow
- Keras
- Deep Learning
- Computer Vision
